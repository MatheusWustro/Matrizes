Matrizes

Introdução as matrizes

Matrizes são um tipo de representção númerica, organizada a partir de uma tabela, onde os números estão organizados em linhas e colunas. Uma matriz é representada a partir de letras maiúsculas acompanhada por índice que mostram a quantidade de linhas e colunas

As linhas são as fileiras horizontais e as colunas as fileiras verticais, já os números dentro das matrizes são os elementos. Então uma Matriz A2x2 significa que ela é uma matriz com duas linhas e duas colunas.

Os elementos genéricos de uma matriz são representados através de letras minusculas, o número da linha e da coluna. Então um elemento a21 é o elemento de uma matriz que está localizado na segunda linha da primeira coluna
